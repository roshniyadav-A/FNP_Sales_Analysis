## 📌 Project Overview

This project is a **Sales Analysis Dashboard built in Excel** for Ferns and Petals (FNP), a gifting company that delivers products for occasions like Diwali, Raksha Bandhan, Valentine’s Day, Birthdays, and Anniversaries.

The objective is to analyze sales performance and customer behavior to generate actionable insights that improve business decision-making and customer satisfaction.

---

## 🎯 Objectives

- Analyze overall **sales and revenue performance**
- Understand **customer purchasing behavior**
- Track **delivery efficiency**
- Identify **top products and top cities**
- Compare **revenue across occasions**
- Study **seasonal sales trends**

---

## 🧹 Data Preparation

The dataset was cleaned and transformed using:

- Power Query for:
  - Removing duplicates
  - Handling missing values
  - Data type corrections
  - Data transformation and shaping

- ⏱️ Average Order & Delivery Time

- 🧑‍🤝‍🧑 Customer Spending Analysis  
- 📦 Top 5 Product Performance  
- 🌆 Top 10 Cities by Orders

---

## 📈 Key Insights

- Highest revenue generated during festive occasions like **Diwali and Valentine’s Day**
- A small number of products contribute to majority of revenue (Pareto Principle)
- Top cities contribute significantly to order volume
- Delivery time remains relatively stable across order sizes
- Strong seasonal trends observed across the year
